What is CSS?

  1. CSS stands for Cascading Style Sheets.

  2. CSS describes how HTML elements are to be desplayed on screen, paper, or in other media.

  3. CSS is not neccesary CSS saves a lot of work. it can control the layout of multiple web pages all at once

  4. External stylesheets are strored in CSS files.

CSS can be addded to HTML document in 3 ways:

  • Inline - by using the style attribute inside HTML elements1
  • internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS file.
  • The mostcommon way to add CSS, is to keep the styles in external CSS files. However, in this
